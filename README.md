<a id="installation"></a>

## üì¶ Installation

1. Clone the repo
   ```sh
   git clone https://github.com/heynzar/kanban-drag-and-drop.git
   cd kanban-drag-and-drop
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the Project
   ```sh
   npm run dev
   ```
